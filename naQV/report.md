## 버전1: KOSPI 지수 vs 주택가격지수 vs 명목GDP

> **"지수(Index) 기준 비교"** - 각각이 기준연도 대비 몇 배 올랐는지

---

### v1_national_1986.png / v1_seoul_1986.png
**1986년 = 100 기준**

| 항목 | 1986년 | 2024년 | 상승률 |
|------|--------|--------|--------|
| KOSPI 지수 | 272.6 | 2,399.5 | **8.8배** |
| 서울 주택가격 | 22.4 | 96.8 | **4.3배** |
| 전국 주택가격 | 26.9 | 93.2 | **3.5배** |
| 명목 GDP | 106조 | 2,557조 | **24배** |

**해석**: 1986년부터 보면 GDP가 24배로 가장 많이 성장. KOSPI는 8.8배, 서울 집값은 4.3배. 명목 GDP는 물가상승이 포함되어 있어서 높게 나옴.

---

### v1_national_1995.png / v1_seoul_1995.png
**1995년 = 100 기준**

| 항목 | 1995년 | 2024년 | 상승률 |
|------|--------|--------|--------|
| KOSPI 지수 | 882.9 | 2,399.5 | **2.7배** |
| 서울 주택가격 | 32.4 | 96.8 | **3.0배** |
| 전국 주택가격 | 41.4 | 93.2 | **2.3배** |
| 명목 GDP | 452조 | 2,557조 | **5.7배** |

**해석**: 1995년 기준으로 보면 서울 집값(3.0배)이 KOSPI(2.7배)보다 약간 더 올랐음. IMF 직전 고점 기준이라 주식이 상대적으로 덜 올라 보임.

---

## PIR (소득대비 주택가격 비율)

> **"내 집 마련에 몇 년 걸리나"** - 연소득 대비 주택가격 배율

---

### pir_chart.png
**PIR: 서울 vs 전국 (2006~2023)**

| 연도 | 서울 | 전국 | 비고 |
|------|------|------|------|
| 2006 | 7.7 | 4.2 | |
| 2008 | 7.5 | 4.0 | 금융위기 |
| 2010 | 7.0 | 4.2 | |
| 2012 | 8.0 | 4.6 | |
| 2014 | 8.5 | 5.0 | |
| 2016 | 8.8 | 5.5 | |
| 2017 | 8.6 | 5.6 | |
| 2018 | 11.4 | 5.5 | 부동산 급등 시작 |
| 2019 | 11.2 | 5.4 | |
| 2020 | 12.5 | 5.5 | 코로나 |
| 2021 | 14.1 | 6.7 | 최고점 |
| 2022 | 15.2 | 6.3 | |
| 2023 | 13.0 | 6.3 | 조정 |

**해석**: 
- 서울: 2018년부터 급등하여 2022년 15.2년(최고)까지 상승 후 2023년 13.0년으로 하락
- 전국: 상대적으로 안정적으로 4~6년대 유지
- 서울은 전국 대비 약 2배 이상의 PIR 유지

**출처**: 국토교통부 주거실태조사, 주택금융공사
