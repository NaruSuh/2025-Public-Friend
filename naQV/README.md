# naQV (naru Query Viewport)

한국은행 경제통계시스템(ECOS) API 및 각종 정치경제 정보 API를 활용한 데이터 분석 프로젝트

## 개요

naQV는 다양한 공공 데이터 API를 통해 정치경제 데이터를 수집하고 분석하는 뷰포트 역할을 합니다.

## 지원 API

- **한국은행 ECOS API**: 경제통계시스템 데이터 조회 및 분석
- (추가 예정) 각종 정치경제 정보 API

## 기능 (예정)

- [ ] ECOS API 연동
- [ ] 경제지표 데이터 조회
- [ ] 데이터 시각화
- [ ] 분석 리포트 생성
